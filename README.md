# 🛍️ Customer Behavior Analysis & Customer Segmentation

A comprehensive **Data Analytics** project that analyzes customer purchasing behavior using **Python, Pandas, Matplotlib, and Seaborn**. The project focuses on data cleaning, feature engineering, exploratory data analysis (EDA), customer segmentation using **RFM Analysis**, churn analysis, and business recommendations.

---

## 📌 Project Overview

Businesses generate large amounts of customer transaction data every day. Analyzing this data helps identify customer purchasing patterns, understand customer behavior, detect churn risks, and create effective marketing strategies.

In this project, customer transaction data is analyzed to:

* Clean and preprocess the dataset
* Perform feature engineering
* Explore customer purchasing behavior
* Segment customers using **RFM Analysis**
* Identify churn risks
* Generate actionable business insights

---

## 🎯 Project Objectives

* Perform data cleaning and preprocessing
* Create useful business features
* Analyze customer demographics
* Study purchasing trends
* Perform customer segmentation
* Analyze customer churn
* Provide business recommendations based on insights

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/your-username/customer-behavior-analysis.git
```

2. Navigate to the project folder

```bash
cd customer-behavior-analysis
```

3. Install the required libraries

```bash
pip install -r requirements.txt
```

4. Open the Jupyter Notebook

```bash
jupyter notebook
```

5. Run all cells to reproduce the analysis.
